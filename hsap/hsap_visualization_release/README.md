# HSAP Visual Simulation

Interactive browser visualization of the Human Social Adaptation Pressure (HSAP) model.

## Quick Start

1. Download or clone this directory
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
3. No server, no Python, no npm, no dependencies required

## Features

- **Phase Map**: Density × threat index trajectory
- **State Map**: Agent rank × energy scatter with cortisol glow
- **Timeline**: Multi-line population, fertility, aggression, and HSAP index
- **Comparison Mode**: Side-by-side scenario comparison
- **PNG Export**: Save timeline screenshots
- **Keyboard shortcuts**: Space (play/pause), Arrow keys (step), R (restart), M (compare), P (PNG)

## Scenarios

| Code | Name | Description |
|------|------|-------------|
| A | Baseline | Standard population dynamics |
| B | Abundance | Low threat, high resources |
| C | Crowded | High density, limited space |
| D | Predation Survival | High predation pressure |
| E | Sink Recovery | Behavioral sink with recovery |
| F | Collapse Risk | Partial collapse trajectory |

## About the Model

HSAP tests whether predator scarcity, low disease pressure, and resource abundance can produce endocrine-linked behavioral shifts in mammalian populations, including reduced male aggression, altered female competition, reproductive restraint, and population stabilization.

Based on: Enholm, J. HSAP: Hormonal-Social Adaptation Pressure Model. 2025.
