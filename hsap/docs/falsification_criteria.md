# Falsification Criteria

The HSAP model is considered weakened or rejected if any of the following hold:

## 1. Simpler models explain the patterns

If null models (N0–N5) fit the observed population/behavior data as well as or better than HSAP with fewer assumptions, HSAP is not needed.

## 2. Endocrine variables add no predictive power

After controlling for density, food availability, predation, disease, and age structure, endocrine proxy variables must improve prediction of aggression, fertility, or population outcomes. If they don't, the hormone layer is ornamental.

## 3. Expected direction fails repeatedly

- Low predation + resource abundance does not reduce male aggression or testosterone proxy.
- Female aggression does not rise under offspring/resource competition.
- Population regulation does not improve compared to null models.
- Reproductive restraint does not increase with density.

Failure in any of these across multiple parameter regimes weakens the hypothesis.

## 4. Extreme parameter dependence

The model only produces HSAP-consistent outcomes with biologically implausible parameter values (e.g., near-zero predation, near-infinite resources, extreme endocrine sensitivity).

## 5. Species-style brittleness

The model works in one life-history configuration (e.g., r-selected rodents) but collapses across plausible alternatives (e.g., K-selected ungulates, primates).

## 6. Calhoun inseparability

If the model cannot reproduce Calhoun-like collapse under high-density/low-space conditions WHILE ALSO producing HSAP regulation under low-density/abundant-resource conditions, the model has not successfully separated the two mechanisms.

## Quantitative thresholds

- A null model achieving within 10% of HSAP's prediction error (RMSE) is considered competitive.
- Any parameter needing values outside 2 standard deviations from documented mammalian ranges is flagged.
- HSAP must outperform the best null model on at least 3 of 5 key metrics (population stability, male aggression, female aggression, fertility, time to collapse).
