# Claim Discipline — Internal Memo

## Core Result Statement (approved)
> "Under the model's assumptions, the HSAP mechanism (low external threat -> male testosterone downshift -> reduced male aggression, elevated female aggression -> reduced fertility) is **simulation-consistent** with population stabilization below carrying capacity. Population trajectories alone are **insufficient** to distinguish HSAP from disease-like regulation; behavioral and endocrine observables are required for falsification."

## Guardrails
| Say | Don't Say |
|-----|-----------|
| simulation-consistent | proves / validates / confirms |
| under model assumptions | biological law / universal mechanism |
| candidate mechanism | the mechanism / the explanation |
| consistent with | demonstrates that |
| suggests / raises the possibility | shows / establishes |
| population trajectories insufficient | HSAP is correct |

## Key Places to Enforce
- **Abstract**: Must use "simulation-consistent" or equivalent hedge.
- **Discussion / Limitations**: Must explicitly state that population data alone cannot distinguish HSAP from disease regulation.
- **B_hsap_abundance**: Must explicitly note that this scenario does not substantially reduce fertility relative to baseline — this is a limitation, not a feature.
